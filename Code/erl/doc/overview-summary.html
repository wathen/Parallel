The modules in this package were written to support examples and homework problems in CpSc 448B. Here's a quick summary:
  • misc An assortment of useful functions.
  • stat Functions for mean and standard deviation.
  • time_it Functions for measuring the time it takes to execute a function, and to log the times of key events during the execution of a function.
  • wlog Extensions to workers to provide event logging for a few of the functions from workers.
  • workers Functions for creating and managing a simple pool of workers.
