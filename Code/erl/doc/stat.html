Description

stat - simple statistical function. Currently, this module provides an accum function for accumulating samples, and functions for computing the mean and standard deviation from such accumulations. I might add more statistical and data analysis functions at a later date, but I don't have any plans to do so immediately.

accum/1

Create a new accumulator using Data If Data is a number, an accumulator is created with that number as it's one sample. If Data is a list of numbers, then an accumulator is created with all of the samples from the list.

mean/1

mean(Stat_acc::X) -> number() | undefined

  • X = accumulator() | number() | [number()]

Compute the mean of the samples in X. If X an accumulator, then the mean is computed for the samples that were used to generate X. If X is a number or a list of numbers, it is used to create an accumulator (see accum/1) and then the mean is calculated. If there are no samples in the accumulator of the list, then the atom 'undefined' is returned.

std/1

Equivalent to std(X, sample).

std/2

std(Stat_acc::X, P_or_S::SampleOrPopulation) -> number() | undefined

  • X = accumulator() | number | [number()]
  • SampleOrPopulation = sample | population

Compute the standard deviation of the samples in X. If X an accumulator, then the standard deviation is computed for the samples that were used to generate X. If X is a number or a list of numbers, it is used to create an accumulator (see accum/1) and then the standard deviation is calculated. The parameter SampleOrPopulation specifies whether to compute a sample or population standard deviation. If you're not sure which you want, you probably want 'sample'. For a 'sample' standard-deviation, there must be at least two samples, or the result is the atom 'undefined'. For a 'population' standard-deviation, there must be at least one sample, or the result is the atom 'undefined'.
